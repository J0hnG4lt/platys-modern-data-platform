# Platform Stack: `modern-data-platform` - v1.10.0

This Platform Stack defines the set of services for a Modern Data Platform, such as

* Kafka
* Spark
* Hadoop Ecosystem
* StreamSets & NiFi
* Zeppelin & Jupyter
* NoSQL

and many others.
